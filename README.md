# Deep Space Explorer

A professional space-themed educational website featuring interactive planet information, space calculators, and a knowledge quiz.

## Features

### Pages
- **Home** - Hero section introducing the space exploration experience
- **About** - Mission statement and project information
- **Planets** - Information on all 8 planets with images and statistics
- **Calculators** - Interactive space calculators
- **Quiz** - Test your space knowledge with 8 questions

### Interactive Elements
- **Custom Space Cursor** - Hollow circle with rotating orbit dot
- **Twinkling Starfield** - 300 animated stars creating an immersive atmosphere
- **Weight Calculator** - Calculate your weight on different planets based on gravity
- **Distance Calculator** - See distances from Earth with light travel time
- **Knowledge Quiz** - 8 multiple-choice questions with scoring

### Design Features
- Immersive cyan and space-blue color scheme
- Orbitron font for futuristic headings
- Smooth page transitions and hover effects
- Responsive design for mobile and desktop
- Sticky navigation bar with glow effects

## How to Use

### Open Locally
1. Double-click `index.html` to open in your browser
2. Navigate using the top menu: Home, About, Planets, Calculators, Quiz
3. Move your mouse to see the custom space cursor
4. Try the calculators and quiz game

### Browser Requirements
- Modern web browser (Chrome, Firefox, Edge, Safari)
- JavaScript enabled
- Internet connection (for loading fonts and planet images)

## Deployment Options

### GitHub Pages (Free)
1. Create a GitHub repository
2. Upload `index.html` and `README.md`
3. Go to Settings > Pages
4. Select main branch and save
5. Your site will be live at: `https://yourusername.github.io/repository-name`

### Netlify (Free)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the `Space Website` folder
3. Your site will be live instantly with a custom URL

### Vercel (Free)
1. Go to [vercel.com](https://vercel.com)
2. Import the project or drag the folder
3. Deploy with one click

## Calculator Details

### Weight Calculator
Calculate what you would weigh on different planets based on their gravity:
- Mercury: 0.38g
- Venus: 0.91g
- Earth: 1.00g
- Mars: 0.38g
- Jupiter: 2.34g
- Saturn: 1.06g
- Uranus: 0.92g
- Neptune: 1.19g

### Distance Calculator
Shows average distance from Earth to each planet in million kilometers, plus:
- Light travel time calculation
- Displays how many minutes light takes to travel the distance
- Uses speed of light: 299,792 km/s
